topics.meneame_tokenizer module¶

Penn Treebank Tokenizer

The Treebank tokenizer uses regular expressions to tokenize text as in PennTreebank. This implementation is a port of the tokenizer sed script written by RobertMcIntyre and available at http://www.cis.upenn.edu/~treebank/tokenizer.sed.

class topics.meneame_tokenizer.TreebankWordTokenizer[source]¶

Bases: nltk.tokenize.api.TokenizerI

The Treebank tokenizer uses regular expressions to tokenize text as in Penn Treebank. This is the method that is invoked by word_tokenize(). It assumes that the text has already been segmented into sentences, e.g. using sent_tokenize().

This tokenizer performs the following steps:

  • split standard contractions, e.g. don't -> do n't

  • treat most punctuation characters as separate tokens

  • split off commas and single quotes, when followed by whitespace

  • separate periods that appear at the end of line

    >>> from topics.meneame_tokenizer import TreebankWordTokenizer
>>> s = '''Good muffins cost $3.88\nin New York.Please buy me\ntwo'''
>>> TreebankWordTokenizer().tokenize(s)
['Good', 'muffins', 'cost', '$', '3.88', 'in', 'New', 'York.', 
'Please', 'buy', 'me', 'two']
>>> s = "They'll save and invest more."
>>> TreebankWordTokenizer().tokenize(s)
['They', "'ll", 'save', 'and', 'invest', 'more', '.']
