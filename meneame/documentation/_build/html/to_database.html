to_database module¶

The aim of this script is to load the articles json files to the couchdbdatabase. The id in the db is set to be the same as the identifier of thearticle on the meneame webiste.

Usage:
to_database.py [–input MENEAME] [–db DATABASE]
Options:
-h, --help show this screen.
--version show version.
--input INPUT input folder.
--db DATABASE output database [default: meneame].
