network package¶

Module contents¶

The aim of this package is to deal with network creation and analysis of the meneame network.

network.network module¶

This module contains the functions for the network analysis and creation.

network.network.community_analysis(graph)[source]¶

Analyze the communities of the input graph.

Parameters:graph – the input igraph object
network.network.create_graph(vertices, edges)[source]¶

Return the graph object, given the edges and vertices collections.

param vertices:collection of vertices, where each element is in the
ormat “username: number_of_comments”
param edges:collection of edges, where each element is in the format (username1, username2): weight, where the weight is the number of articles in which the two users have commented together :returns: the igraph object

The creation of the graph is done only on the final step of the function, due to the way igraph deals with the edges. Infact, building first the graph and then the edges would result in a very inefficient code, as the edges are re-indexed every time a new edge is added.

network.network.save_log_histogram(x, y, title, xlabel, ylabel, filename, image_folder)[source]¶

Create a loglog histogram from the input x and y and saves it to file. Each bin has an unitary size.

Parameters:
  • x – bin positions
  • y – count values
  • title – title of the histogram
  • xlabel – label of the x axis
  • ylabel – label of the y axis
  • filename – name of the saved file
